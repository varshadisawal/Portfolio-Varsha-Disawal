Project Overview

  • Surf Forecast is a web app which displays wind and wave forecasts for customizable locations. It features a visually appealing charts for both Wind speed and Wave height which are crucial factors determining quality of Surf. There is also functionality to add new locations based on latitude and longitude which give enables users to create dashboards for their home surf spot.
  • The website is a dynamic web app built with Microsoft .NET MVC Framework with an Azure SQL instance as a backend. Data is sourced from the Windy.com API via JavaScript and saved to the database using the .NET MVC Entity Framework. The Windy API provides us with weather model forecasts in JSON format. We leverage Chart JS to create the line charts for each forecast type.
