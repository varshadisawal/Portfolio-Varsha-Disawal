Project Overview

  • The police fatal shootings dashboard analyzes data on fatal police shootings in the United States from 2015-2022. The raw data comes from the Washington Post's database of fatal police shootings in the US. This project focused on visualizing insights from the data to understand trends and patterns in fatal police shootings over time. The interactive Tableau dashboard allows users to filter by year, location, demographics, and other variables.
  • Key visualizations include :

    • a - Trends in fatal shootings by year and month

    b - Fatal shootings by state

    c - Demographic breakdown of victims (race, age, gender)

    d - Comparison of victim demographics vs. general population

    e - Mapping where fatal shootings occurred

  • Building this dashboard required data cleaning and transformation in Python before visualization in Tableau. Skills demonstrated include aggregating large datasets, creating calculated fields and parameters, design principles for effective visualizations, and storytelling with data. This project highlights proficiency in turning raw data into actionable insights. The ability to analyze police shooting trends has significance for policy discussions on law enforcement practices and reform.
